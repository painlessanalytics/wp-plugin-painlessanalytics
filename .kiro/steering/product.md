# Product Overview

Painless Analytics is a WordPress plugin that integrates simplified web analytics tracking into WordPress sites.

## Core Purpose
- Provides easy integration with the Painless Analytics web analytics service
- Focuses on metrics that matter most without tracking PII or using cookies
- Offers a simple setup process using link codes from painlessanalytics.com

## Key Features
- Asynchronous JavaScript tracking code injection
- Link code-based configuration (connects to Painless Analytics API)
- Optional admin area tracking (separate or combined with public tracking)
- Automatic redirect to settings page on plugin activation
- No cookies, no PII tracking

## External Service Integration
- Communicates with api.painlessanalytics.com for link code validation
- Retrieves tracking API URLs based on link codes
- Requires active Painless Analytics account (free or paid)
- See: https://www.painlessanalytics.com/company/legal/privacy/
